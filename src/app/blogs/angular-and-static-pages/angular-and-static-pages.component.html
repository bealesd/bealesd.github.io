Generating Static Pages in Angular with Scully

Overview

Static websites are great, here are some of the reason why:

  • simple to setup
  • easy to host
  • fast load times
  • more secure

For a simple site, like my blog, static sites are ideal. However, I've written my blog in Angular, which is dynamic. Welcome Scully. Scully will take yor Angular website, find all your routes, and generates a static website.

Pre requisites

  • An Angular project with some URL routes.
  • A local webserver - i.e. Web server for Chrome, Python 3 http server, etc.

Installing Scully

To install Scully run this command in the root of your Angular project.

    
        ng add @scullyio/init

A Scully config file will be created in the project root scully.<projectName>.config.ts. 

    
        import { ScullyConfig } from '@scullyio/scully';
        export const config: ScullyConfig = {
          projectRoot: "./src",
          projectName: "blog",
          outDir: './docs',
          routes: { }
        };
